Additional options that may be passed to sizeScale().

Type declaration

  • Optional prefix?: string

    An optional prefix that you can add to scale keys. Instead of each key being just a number, you can use a prefix to add differentiating names to each scale position, such as "box0" or "space3".

  • Optional step?: number

    The amount of units that each interval will increase by.
